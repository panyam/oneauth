API Token

The Auth Server also supports programmatic access via POST /api/token. This is the same OAuth2 password grant used by CLI tools and mobile apps. 

POST /api/token
{"grant_type":"password","username":"...","password":"..."}

Returns access_token + refresh_token as JSON. See test_08 for examples.
