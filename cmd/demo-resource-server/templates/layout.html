Resource Server Status

What is this?

This is a Resource Server — a service that protects access to resources (APIs, WebSockets, files, etc.) by validating JWTs. It reads signing keys from a shared PostgreSQL signing_keys table. It has no direct connection to any app — it trusts tokens purely because it can verify their signatures against the KeyStore.

In production, this could be an API gateway, a WebSocket server, a microservice, or any endpoint that needs to verify "is this request authorized?"
